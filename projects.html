Sensing and Estimating in Robotics

Orientation Tracking

View Report

Developed algorithms to estimate the 3-D orientation of a rotating body using inertial measurement unit (IMU) data. Implemented IMU calibration, quaternion-based motion and observation models, and projected gradient descent optimization to refine orientation estimates. Validated against VICON ground-truth data and generated panoramic images by stitching camera frames using the estimated orientations.

  • Python (NumPy, SciPy, JAX, Matplotlib)
  • Transforms3D & SciPy Spatial Transformations
  • Quaternion Kinematics & Optimization
  • Sensor Calibration (IMU, Camera)
  • Data Synchronization & Visualization

LiDAR-Based SLAM

View Report

Implemented a full SLAM pipeline using encoder and IMU odometry, 2D LiDAR scans, and RGB-D measurements from a differential-drive robot. Developed motion models and scan-matching with ICP to refine odometry, built an occupancy grid map of the environment, and applied factor-graph optimization with GTSAM for loop closure. Extended mapping with RGB-D texture projection to create a colored floor map.

  • Python, NumPy, Open3D, OpenCV, Matplotlib
  • ICP scan matching
  • Occupancy grid mapping
  • Factor graph optimization (GTSAM)
  • Texture mapping with RGB-D

Visual-Inertial SLAM

View Report

Implemented a Visual-Inertial Simultaneous Localization and Mapping (SLAM) system using an Extended Kalman Filter (EKF). Integrated IMU kinematics for prediction and stereo camera observations for correction to estimate robot trajectories and landmark positions. Developed modules for IMU-based localization, landmark initialization/mapping, and feature detection/tracking, and fused them into a full SLAM pipeline that reconstructs 2D/3D trajectories with visualized maps.

  • Python (NumPy, SciPy, OpenCV, Matplotlib)
  • Extended Kalman Filter
  • SE(3) Kinematics
  • Stereo Vision & Feature Tracking
  • Sparse Matrix Optimization

Planning and Learning in Robotics

Dynamic Programming: Door & Key Navigation

View Report

Implemented dynamic programming to solve autonomous navigation in the MiniGrid Door & Key environment. Designed motion models and cost functions to optimize action sequences, enabling an agent to pick up keys, unlock doors, and reach goals with minimal energy. Developed policies for both fixed known maps and generalized strategies for random 10×10 environments with varying key, door, and goal placements.

  • Python
  • Dynamic Programming
  • Markov Decision Processes (MDP)
  • MiniGrid Gymnasium
  • Autonomous Navigation

3D Motion Planning

View Report

Implemented collision checking, search-based planning (A*), and sampling-based planning (RRT, RRT*) in 3D environments. Compared performance across algorithms in terms of path quality, runtime, and efficiency in navigating complex obstacle spaces.

  • Python
  • A*
  • RRT / RRT*
  • 3D Collision Checking
  • Search-Based Planning
  • Sampling-Based Planning

Safe Trajectory Tracking

View Report

Developed control policies for a differential-drive robot to track reference trajectories while avoiding obstacles under motion uncertainty. Implemented receding-horizon Certainty Equivalent Control (CEC) for deterministic approximation and Generalized Policy Iteration (GPI) for stochastic optimization with discretized state and control spaces. Evaluated performance on tracking accuracy, collision avoidance, and computational efficiency.

  • Python
  • Certainty Equivalent Control (CEC)
  • Generalized Policy Iteration (GPI)
  • Dynamic Programming
  • Trajectory Tracking
